# Caso de Uso: Fazer Login (CDU02)

## Atores Envolvidos

* Usuário

## Pré-condições

* O usuário possui cadastro no sistema.

## Pós-condições

* O usuário é autenticado e tem acesso às funcionalidades do sistema.

## Fluxo Principal

1. O usuário informa login e senha.
2. O sistema valida as credenciais.
3. O sistema concede acesso à conta do usuário.

## Fluxo de Exceção

* **2a.** Caso o login ou a senha sejam inválidos, o sistema informa o erro e solicita uma nova tentativa.
