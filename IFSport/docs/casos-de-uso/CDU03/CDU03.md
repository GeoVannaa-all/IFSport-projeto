# Caso de Uso: Editar / Visualizar Perfil (CDU03)

## Atores Envolvidos

* Usuário

## Pré-condições

* O usuário deve estar logado no sistema.

## Pós-condições

* Os dados do perfil são exibidos ou atualizados com sucesso.

## Fluxo Principal

1. O usuário acessa a opção de perfil.
2. O sistema exibe as informações cadastradas.
3. O usuário altera os dados desejados.
4. O sistema salva as alterações.

## Fluxo de Exceção

* **3a.** Caso algum dado informado seja inválido, o sistema solicita a correção antes de salvar as alterações.
